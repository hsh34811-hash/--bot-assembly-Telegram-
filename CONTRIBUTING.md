# 🤝 دليل المساهمة

شكراً لاهتمامك بالمساهمة في المشروع! كل مساهمة مرحب بها.

---

## 📋 قواعد المساهمة

- احترم أسلوب الكود الموجود
- اكتب كود نظيف وقابل للقراءة
- لا ترفع أي توكنات أو مفاتيح حساسة
- اختبر التغييرات قبل رفعها

---

## 🚀 خطوات المساهمة

**1. Fork المشروع**

**2. أنشئ فرع جديد**
```bash
git checkout -b feature/اسم-الميزة
```

**3. ارفع التغييرات**
```bash
git commit -m "إضافة: وصف التغيير"
git push origin feature/اسم-الميزة
```

**4. افتح Pull Request**

# 🤝 Contributing Guide

Thank you for your interest in contributing!

## Steps

1. Fork the repository
2. Create a new branch: `git checkout -b feature/your-feature`
3. Commit your changes: `git commit -m "Add: description"`
4. Push to the branch: `git push origin feature/your-feature`
5. Open a Pull Request

## Rules

- Follow existing code style
- Never commit tokens or sensitive keys
- Test your changes before submitting
